## Overview

Unseen provides a centralized space to create, organize, and collaborate on your project documentation. You build structured docs sites with Markdown, version history, and powerful search. Teams use Unseen to maintain living documentation that stays current across projects.

<Callout kind="info">
Unseen integrates seamlessly with your Git workflow, so you version control docs alongside code.
</Callout>

## Key Features

Unseen offers essential tools for documentation management.

## Benefits for Teams and Projects

You streamline documentation workflows, reducing silos and outdated info. Teams collaborate in real-time, with permissions and review workflows. Projects benefit from consistent styling and easy navigation, improving onboarding speed by up to 50%. Use Unseen to scale docs as your project grows, from open-source repos to enterprise knowledge bases.

## Quick Start

Get up and running with Unseen in minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Create a free account at https://app.unseen.com/signup.
  </Step>
  <Step title="Create Space" icon="folder-plus">
    Click "New Space" and name it for your project, like "MyApp Docs".
  </Step>
  <Step title="Add First Page" icon="file-plus">
    Create `introduction.mdx` and start writing.
  </Step>
</Steps>

Here is a sample page structure you can copy:

<CodeGroup tabs="Markdown,HTML">
  ```markdown
  ---
  title: My Page
  description: Page summary
  ---

  ## Overview

  Welcome to your docs.
  ```
  ```html
  <!DOCTYPE html>
  <html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    <h2>Overview</h2>
    <p>Welcome to your docs.</p>
  </body>
  </html>
  ```
</CodeGroup>
